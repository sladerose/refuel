  <action>
    Create a new file `refuel/Models.swift` and define two SwiftData models:
    1. `Station`:
       - `id`: UUID (unique)
       - `name`: String
       - `address`: String
       - `latitude`: Double
       - `longitude`: Double
       - `openingHours`: String?
       - `services`: [String]?
       - `lastUpdated`: Date?
       - `@Relationship(deleteRule: .cascade)` `prices`: [FuelPrice]
    2. `FuelPrice`:
       - `id`: UUID (unique)
       - `grade`: String (e.g., "91", "95", "Diesel")
       - `price`: Double
       - `timestamp`: Date
       - `@Relationship` `station`: Station?
    
    Ensure models use the `@Model` macro and appropriate SwiftData annotations.
  </action>
