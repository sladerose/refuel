  <name>Task 1: Implement ValueAnalyticsService and update Station model</name>
  <files>refuel/Models.swift, refuel/ValueAnalyticsService.swift, refuelTests/ValueAnalyticsTests.swift</files>
  <behavior>
    - Station should have a persistent `zScore: Double?` and a computed `ragStatus` (enum: exceptional, good, average, expensive, avoid).
    - ValueAnalyticsService should calculate mean and standard deviation of minimum prices across a collection of stations using Accelerate (vDSP).
    - It should update each station's `zScore` and correctly map Z-scores to RAG status:
      - Z < -1.5: Exceptional (Dark Green)
      - Z < -0.5: Good (Green)
      - -0.5 <= Z <= 0.5: Average (Yellow)
      - Z > 0.5: Expensive (Orange)
      - Z > 1.5: Avoid (Red)
  </behavior>
  <action>
    1. Update `Station` model in `Models.swift`:
       - Add `var zScore: Double?`.
       - Add `enum RAGStatus: String, Codable, CaseIterable` with colors.
       - Add computed property `var ragStatus: RAGStatus`.
    2. Create `refuel/ValueAnalyticsService.swift`:
       - Use `import Accelerate`.
       - Implement `func calculateAnalytics(for stations: [Station])`.
       - Handle edge case where stdDev is 0 (all prices same) or empty list.
    3. Create `refuelTests/ValueAnalyticsTests.swift` to verify math and RAG mapping.
  </action>
  <verify>
    <automated>swift test --filter ValueAnalyticsTests</automated>
  </verify>
  <done>Station model updated and analytics math verified with tests.</done>
