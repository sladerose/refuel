### SocialSyncManager.publicDB is a computed property

`refuel/SocialSyncManager.swift` lines 57-59:

```swift
private var publicDB: CKDatabase {
    CKContainer(identifier: Self.containerID).publicCloudDatabase
}
```

This is a computed property — NOT a `lazy var` or stored property. This pattern avoids Swift 6 async concurrency crashes that occur when a stored `CKDatabase` is accessed across actor boundaries.
